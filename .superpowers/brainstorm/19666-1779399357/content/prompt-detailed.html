Six directions — light + dark, with what each can do

Named, cleaned up, and shown in both themes (Caldera's Mineral dark + bone light). Under each: the features it unlocks. Pick the one to take all the way — or mix two.

1 · Cockpit

all telemetry in a persistent strip; the prompt is one pure glyph
Dark
caldera-console  main ●2 ↑1✓ 1.2s  09:16
❯ zig build test
❯
Light
caldera-console  main ●2 ↑1✓ 1.2s  09:16
❯ zig build test
❯
Strip: cwd · branch · dirty · ahead/behind · exit · duration · clock Live — refreshes between prompts, not frozen text Contextual chips: venv, SSH host, jobs appear only when present Docks top or bottom · prompt stays a bare ❯

2 · Strata

every command + output is a quiet block
Dark
❯ git status
  working tree clean
❯ caldera · main ●2 · 09:16
Light
❯ git status
  working tree clean
❯ caldera · main ●2 · 09:16
Blocks built from the OSC 133 marks Caldera already parses Failed run → red edge · running → animated edge Jump prompt-to-prompt; select a whole block at once Long output folds to a summary line (later)

3 · Pulse

shows only what's true right now — grows when it must
Dark
caldera ❯  — clean
caldera main ●3 ❯  — dirty
caldera ✗127 · 1.2s ❯  — failed
Light
caldera ❯  — clean
caldera main ●3 ❯  — dirty
caldera ✗127 · 1.2s ❯  — failed
Segments appear by relevance: dirty, exit, duration>1s, rebase/merge state, venv Glyph tints red after a failure Transient — past prompts collapse to a dim one-liner Clean repo = the calmest possible prompt

4 · Halo

Caldera draws the context line — so it's genuinely live
Dark
❯ git status · clean
caldera-console⎇ main ●2⠋building 4.2s
❯
Light
❯ git status · clean
caldera-console⎇ main ●2⠋building 4.2s
❯
Spinner + live duration while a command runs git state refreshes as files change, between keystrokes Smooth color, real graphics — not ANSI text Pure ❯ underneath → still works in any terminal · room for an inline ask-AI affordance
