1 · Quiet

A dim context whisper, one line above a clean prompt — and it's transient: once a command runs, that whisper line disappears. Scrollback is just ❯ commands and output, nothing else. — my pick.

Integrations: appear as dim tokens on the whisper line, and only while live — caldera · main ●2 · zig 0.16 · ci building · k8s prod — gone the moment they're idle. Loud is impossible; the line is always one dim sentence.

2 · Right-whisper

One line. You type at the far left; context is a faint cluster pinned to the right edge — so far from your cursor it never crowds you, but glanceable any time.

Integrations: extra tokens just extend the right cluster, all in the same faint grey. The right edge becomes a quiet status column down your whole scrollback.

3 · Sigil

The everyday prompt is one character — ◆ — its color the only signal (mineral clean, amber dirty, red failed). The full context is one keypress away, shown below expanded, then folds back.

Integrations: live in the revealed line — as many as you want, hidden by default. The resting prompt never grows, no matter how many tools you wire in.
