Calm = just the anchor (cwd) + git + toolchain. Modules stay silent until they have something to say. A failed command reddens the ◆; a long run shows its duration; CI only appears while a run exists.

Why it extends — anatomy of a module

Every item in the Seam is the same tiny unit: a state dot · a label · a value. One consistent shape means ten integrations still read as one calm row, never a sticker bomb. Three ways a module gets there:

Core

Shipped in Zig — cwd, git, exit, duration, toolchain. Fast, always available, zero config.

Declared

You add one in config.zon: a command to run + when to refresh. The Starship-custom-module idea, baked in.

Integration

A plugin pushes live values over Caldera's IPC bridge — CI, k8s, Docker, cloud, your coding agents. Updates between keystrokes.

Order, visibility, and which theme color each state uses — all config.zon. The Seam is drawn by Caldera (not the shell), so it can refresh live, animate a spinner, and stay pixel-clean — while the ◆ input line below stays a pure, portable prompt.
