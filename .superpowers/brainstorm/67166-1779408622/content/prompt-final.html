The Caldera prompt — the design

Synthesised from the whole brainstorm: a low-profile agent-style input, written in Zig, shipped via the shell integration. Two lines while live, then it collapses — scrollback stays clean. A mineral edge marks where you type. Calm, static, pure ANSI text — works in any terminal.

Live — what it looks like as you work

Past commands collapse to a bare ❯ command — only the active prompt is the two-line block with the mineral edge. Git and last-result sit on the context line; the input line stays clean.
