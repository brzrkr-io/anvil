# Anvil Keymap Cheatsheet

Also available in-app: press **⌘/** or run the palette command **"Help: Keyboard Shortcuts"**.

## General
| Shortcut | Action |
| --- | --- |
| ⌘K | Command palette |
| ⌘P | Find file |
| ⌘, | Settings |
| ⌘/ | Keyboard shortcuts |
| ⌘+ / ⌘− / ⌘0 | Zoom in / out / reset |

## Files & Editor
| Shortcut | Action |
| --- | --- |
| ⌘O | Open file |
| ⌘⇧O | Open folder / Go to symbol |
| ⌘S | Save |
| ⌘W | Close tab |
| ⌘⇧T | Reopen closed tab |
| ⌘E | Recent files |
| ⌘F | Find / replace |
| ⌥B | Toggle blame |
| ⌘⌥K | Toggle bookmark |
| ⌘⌥J / ⌘⌥⇧J | Next / previous bookmark |
| ⌘⌥← / → | Navigate back / forward |
| F12 | Go to definition |
| F2 | Rename symbol |
| ⇧F12 | Find references |
| ⌘. | Code actions |
| ⇧⌥F | Format document |

## Terminal
| Shortcut | Action |
| --- | --- |
| ⌘T | New terminal |
| ⌘D | Split / unsplit |
| ⌘J | Toggle bottom terminal |
| Ctrl+` | Quake drop-down terminal |

## Workspace
| Shortcut | Action |
| --- | --- |
| ⌘B | Toggle explorer sidebar |
| ⌘⇧B | Toggle activity rail |
| ⌘\ | Split workspace panes |
| ⌘⇧⏎ | Zoom pane |
| ⌘N | New window |
| ⌘. | Zen / terminal mode |

## Markdown & Agent
| Shortcut | Action |
| --- | --- |
| ⌘⇧V | Toggle Markdown preview |
| ⌘I | Ask agent |
| ⌘⌥E | Explain selection |

> Keybindings are remappable in Settings, and presets (VS Code / Zed) are available via the palette command **"Keybindings: Apply Preset…"**.
