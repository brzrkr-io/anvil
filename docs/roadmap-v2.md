6. [x] Markdown preview — `MarkdownPreview.svelte` (marked, palette-styled), ⌘⇧V / palette toggle; renders the **live editor buffer** (unsaved edits included) via the `editorLive` store, falling back to disk + poll for non-active files.

10. [x] Large-file mode — files > 2 MB load plaintext with LSP / syntax / swatches / indent-markers / minimap / inlay disabled to stay responsive.
