Design notes

  • CONTEXT: env-tint dot encodes cluster risk. Prod → ATTENTION amber; staging → INFO_TEAL; dev/local → ALLOY. Hidden when kubectl absent.
  • REPO+GIT: merged from two sections. Saves one header row. New SectionId::RepoGit variant.
  • CI: replaces LAST RUN. Status glyph + branch + duration on row 1. Open PR count on row 2 only when > 0. Cmd-click opens PR URL. Requires gh-ci poller.
  • AGENTS: unchanged semantics. Idle state uses VERIFIED green dot, not ALLOY.
  • SYSTEM: demoted to single compact row. Clock removed (lives in bottom-bar already).
  • HUD surface: GRAPHITE at 88% alpha. 1px left hairline at tones.edge.
  • Collapse state: u8 bitmask on App. Header row always visible. No force_full_redraw from HUD ticks.
