## Pivot context

- nvim-as-editor (BR3 + BR4 + BR5) is **shipped** but **deprecated**. Code
  stays for now under `crates/anvil-editor/src/nvim/`. NE15 removes it.
- User reasoning (paraphrased from `2026-05-25-native-editor-roadmap.md`):
  Anvil should own the editor. Native text-editing surface rendered with the
  same Metal cell grid that paints the terminal. No more RPC roundtrip for
  AI interactions. Caldera agents call an in-process buffer API.
- The `Cmd+E` "New Editor Pane" still spawns `nvim --listen`. NE4 will
  re-point this to a native `EditorPane` spawned in the pane tree.
