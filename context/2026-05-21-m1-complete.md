  Anvil Basin mark as the macOS Dock icon.
- **M1 done** — a genuinely usable single-pane GPU terminal:
  - `src/terminal/` — VT/ANSI parser (Williams DFA), grid, alternate screen,
    deep scrollback (ring of trimmed rows, `default_capacity = 100_000`).
  - `src/pty/` — `openpty` + fork/exec the login shell; read/write/resize.
  - `src/render/` — CoreText glyphs → BGRA bitmap → Metal texture, drawn by a
    runtime-compiled MSL shader (no offline metal toolchain needed).
  - `src/app/keys.zig` — keyboard → terminal-byte encoding.
  - `src/main.zig` — integration: PTY reader thread + a 60 Hz tick, keyboard
    and scroll input, window resize.
  - OSC 133 semantic prompt marks are recorded (`Terminal.promptMarks()`).
  - 102 unit tests pass. Verified end-to-end: spawns zsh, renders colored
    output, typing / scroll-wheel history / resize all work.
