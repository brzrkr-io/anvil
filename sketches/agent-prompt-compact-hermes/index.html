Less board. More prompt.

Hermes’ own CLI prompt is intentionally tiny: a prompt symbol, then state prefixes only when needed. This pass keeps that discipline. Bracket meters become compact side/status chips, not giant dashboards. Default prompt stays one line; expanded telemetry is one short line above or below.
