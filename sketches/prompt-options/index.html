Anvil prompt visual companion / 20 directions

Prompt shapes that stay operator-grade.

Twenty compact prompt treatments using the Anvil palette. The goal is not decorative powerline chrome; it is a smarter shell affordance where glyphs and color encode repo, failure, agent, execution, and evidence state.
