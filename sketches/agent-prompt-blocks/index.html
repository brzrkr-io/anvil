- Removed default box-draw frame
- Kept the regular prompt left-aligned
- Moved activated agent prompt / telemetry to the right
- Replaced frame hierarchy with bars, blocks, rails, and meters
- Preserved semantic colors: violet agent, mineral trace, ember execution, amber approval, green evidence, red blocked
