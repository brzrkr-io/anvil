Make the hook hit.

The prompt grammar is now simple: in agent mode, the live caret is ⌁. Brackets, pipes, loading bars, and state rails can surround it, but the input mark itself is the hook/pulse glyph. This pass pushes the hacker/operator/Hermes/Honcho vibe harder without turning the prompt into a whole dashboard.

fixed rule

Shell mode:
  repo branch › command

Agent mode:
  ⌁ ask agent

Good:
  [hermes|ctx ▮▮▮░░] ⌁ ask
  [⌁|73|141k] ⌁ continue
  │ ctx ▮▮▮▮░░ │ tok ▮▮░░░ │
  ⌁ route task

Bad:
  ⚡ ask
  ⌁ › ask
  ⌁ ⚡ ask
