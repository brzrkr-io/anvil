agent-mode prompt revisionrule: bolt replaces arrowHermes / Honcho / operator console
[hacker · hermes · honcho · compact cyberpunk]

Bolt is the prompt.

The earlier options kept thinking in shell-arrow grammar. Agent mode should feel like an activated execution channel: no orange `›`, no arrow-plus-bolt combo. The lightning bolt is the input prompt. Hook/notch become optional identity or mode markers; the bolt is where the user types.

hard rule

Bad:
  ⌁ › ask
  ⌁ ⚡ › ask

Good:
  ⚡ ask
  ⌁ ⚡ ask
  [⌁ ctx 73] ⚡ ask

Meaning:
  ⌁ = agent signal / attached channel
  ⌙ = Anvil scoped/gated mode
  ⚡ = live agent prompt caret

new grammar

Shell mode:
  repo branch › command

Agent mode:
  ⚡ ask agent

Agent mode with identity:
  ⌁ ⚡ ask agent

Agent mode with telemetry:
  [⌁ ctx ▮▮▮▮░░ 73 · tok 141k]
  ⚡ ask agent

No prompt should render `›` in agent mode.

recommendation

Use:
  ⌁ ⚡

Expanded:
  [⌁ ctx ▮▮▮▮░░ 73 · tok 141k]
  ⚡

Use ⌙ only for Anvil scoped/gated states:
  [⌙ gated · repo:anvil]
  ⚡ confirm edit
