My recommended direction.

Separate agent prompt

Clear, but risky. It fragments the command surface and can feel like another product living inside the terminal.

  • Use only in expanded mode.
  • Good for command decks or handoff workflows.

Signify inside prompt

Best default. The operator prompt stays trusted; glyph/color state adds context without stealing the line.

  • Matches option 2.
  • Works for Hermes/Caldera status.

Hybrid companion

Recommended. Normal thin prompt by default; toggle a violet agent drawer/rail for multi-agent details, claims, approvals, and evidence.

  • Most Anvil-native.
  • Scales to caldera-os and other agents.
