## Recommendation

Use `⌁` as the default agent prompt symbol.
Use `⌙` as the Anvil-branded / scoped / policy / gated alternate.
Use bracketed segmented meters only in expanded mode or state rows, not on every single prompt line.
Use violet for meters and orange/ember for the live prompt arrow.
