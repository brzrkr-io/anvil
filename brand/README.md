# Anvil Brand Assets

This directory holds portable brand assets derived from the Anvil brand exploration.

Use these files as source assets for apps, CLIs, docs, decks, and product repos. Standalone repos should copy the assets they need; they must not depend on `/Users/pjanderson/projects/caldera/caldera-os` at runtime.

## Files

- `mark.svg` - Basin mark, black/current ink.
- `mark-inverted.svg` - Basin mark, bone/current light ink.
- `lockup.svg` - Basin mark plus `Anvil` wordmark.
- `app-icon.svg` - rounded-square app icon.
- `avatar.svg` - circular GitHub/org avatar.
- `favicon.svg` - compact favicon treatment.
- `tokens.css` - CSS custom properties for the Mineral palette and semantic states.
- `tokens.json` - machine-readable token set.

Read `../BRAND.md` before using or modifying these assets.
